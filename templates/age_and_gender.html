WHAT IS CNN?

The full form of CNN is convolutional neural network and using this neural network we can process our data which is having a shape like 2D matrix like images. Most probably the convolutional neural network is mostly being used for image detection and image classification type. We know that the images are represented in the form of 2D matrix of pixels so that we can use convolutional neural network to recognize or classify the image. Thus, suppose we have to recognize which are the objects and which are the faces then we are using the convolutional neural network to identify it. In machine learning or deep learning the convolutional neural network is a class of deep neural network used for analyzing images.

WORKING OF CNN

There are mainly four layers in the convolutional neural network: 1. Convolutional layer 2. Activation layer 3. Pooling layer 4. Fully connected layer. The convolutional layer of CNN is the most important layer as it does the work of taking the input and then connecting it with the further layer. Convolutional layer is considered as the core building block as it does the work of dead lifting. This layer takes the input of data with the help of a filter or a kernel and after the process of filtering this image or data is sent further for processing.

The second layer is said to be the activation layer which is applying the ReLu (Rectified Linear Unit). This second layer does the work of increasing the non-linearity in the CNN. This layer also helps the model to decide if a neuron would fire or not.

The third layer in the convolutional neural network is the pooling layer which involves down sampling of features. This layer does the work of progressively reducing the spatial size of the representation. It does the work of reducing the amount of parameters and computation in the network. This layer operates independently on each feature map.

The final layer is the fully connected layers and it involves flattening. It takes the data from the third layer i.e. the pooling layer mixes the whole matrix part and after the mixing whatever the final output is being obtained it is provided to the model and the final answer is being predicted on the basis of that.

Now let’s understand how CNN recognizes a face: The pixels from the image are fed to the convolutional layer that performs the convolution operation. This results in formation of a convolved map. The convolved map is then applied to a ReLU function which generate a rectified feature map. Now the image is processed with multiple convolutions and ReLU layers for locating the features. Different pooling layers with various filters are used to identify specific parts of the image. The pooled feature map is flattened and fed to a fully connected layer to get the final output and in other words we can say that the any portion of the face is taken it is being matched with the images given for training then where most of the things are being match it predicting as male or female.
